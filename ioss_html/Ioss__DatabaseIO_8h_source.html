463  /*!
464  * The overlay_count specifies the number of restart outputs
465  * which will be overlaid on top of the currently written
466  * step before advancing to the next step on the restart
467  * database.
468  *
469  * For example, if restarts are being output every 0.1
470  * seconds and the overlay count is specified as 2, then
471  * restart will write time 0.1 to step 1 of the database.
472  * It will then write 0.2 and 0.3 also to step 1. It will
473  * then increment the database step and write 0.4 to step 2;
474  * overlay 0.5 and 0.6 on step 2... At the end of the
475  * analysis, assuming it runs to completion, the database
476  * would have times 0.3, 0.6, 0.9, ... However, if there
477  * were a problem during the analysis, the last step on the
478  * database would contain an intermediate step.
479  *
480  * The cycle_count specifies the number of restart steps
481  * which will be written to the restart database before
482  * previously written steps are overwritten.
483  *
484  * For example, if the cycle count is 5 and restart is written every 0.1
485  * seconds, the restart system will write data at times 0.1, 0.2, 0.3,
486  * 0.4, 0.5 to the database.
487  *
488  * It will then overwrite the first step with data from time 0.6, the
489  * second with time 0.7. At time 0.8, the database would contain data at
490  * times 0.6, 0.7, 0.8, 0.4, 0.5. Note that time will not necessarily be
491  * monotonically increasing on a database that specifies the cycle
492  * count.
493  *
494  * The cycle count and overlay count can both be used at the same time
495  * also. The basic formula is:
496  *
497  * db_step = (((output_step - 1) / overlay) % cycle) + 1
498  *
499  * where "output_step" is the step that this would have been on the
500  * database in a normal write (1,2,3,....) and "db_step" is the step
501  * number that this will be written to.
502  *
503  * If you only want the last step available on the database,
504  * use set_cycle_count(1)
505  */
506  void set_cycle_count(int count) const { cycleCount = count; }
507  int get_cycle_count() const { return cycleCount; }
508  void set_overlay_count(int count) const { overlayCount = count; }
509  int get_overlay_count() const { return overlayCount; }
